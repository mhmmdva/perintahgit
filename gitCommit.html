Materi Git Commit!

git config --global user.name "mhmmdva"

git config --global user.email "vallenleyss@gmail.com"

git commit -a -m (-am) "komentar untuk pemberian history" jika suatu file sudah modified pada saat pengecekan status, maka kita bisa langsung memasukkkan perintah seperti itu untuk mempercepat pekerjaan dengan artian -a untuk menambahkan dan -m untuk commit

git commit : Sebelum melakukkan commit , kita harus config atau menjelaskan siapa diri kita ke dalam Git dengan menggunakan perintah :

git commit -m "komentar untuk pemberian history" untuk melakukan commit atau menentukan bahwa code yang sudah kita buat itu akan dimasukkan ke dalam git
