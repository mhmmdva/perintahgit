Perintah Rebase

Rebase ini adalah suatu perintah untuk mensejajarkan branch lain dengan branch master, dan fungsinya ini adalah untuk menghindari adanya Three-way atau conflict pada git saat ingin melakukan push.

jika ingin melakukan rebase ini, maka kita harus melihat terlebih dahulu pada graph apakah branch yang sudah kita buat ingin kita lakukan rebase agar terjadinya fast-forward untuk menghindari adanya conflict

  • git checkout (nama_branch_lain)
  • git rebase (nama_branch_master)
